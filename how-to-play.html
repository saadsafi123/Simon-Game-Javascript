How to Play Simon Game

Objective

The goal of the Simon Game is to repeat the sequence of colors that the game plays. The sequence gets longer with each round, and you must remember and repeat the sequence to advance to the next level.

How to Play

  1. Press any key or the "Start Game" button to begin..
  2. The game will light up one of the four colored buttons and play a sound. Remember the color and the sound
  3. After the first button lights up, you need to click on that same button to repeat the sequence.
  4. With each round, the game will add one more button to the sequence. Your task is to remember the entire sequence and repeat it in the correct order by clicking on the buttons
  5. If you click the wrong button or forget the sequence, the game is over.
  6. Try to achieve the highest score possible by remembering longer and longer sequences.
